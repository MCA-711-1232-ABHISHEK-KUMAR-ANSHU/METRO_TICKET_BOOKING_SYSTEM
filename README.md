## Project Description

This project has been developed as part of the academic requirements for the attainment of the **Master of Computer Applications (MCA)** degree.

The project demonstrates the conceptual implementation of a **Metro Ticket Booking System** developed using the **Flask Web Framework**. The system provides a complete digital solution for metro ticket booking, QR code generation, ticket validation, passenger management, route management, fare calculation, cancellation processing, and administrative monitoring.

The application enables passengers to register, log in, book metro tickets, download ticket PDFs, scan QR codes for ticket verification, and view travel history. Administrators can manage stations, routes, fares, bookings, cancellations, and system reports through a dedicated dashboard.

---

## Technology Stack & Deployment

The project has been developed using the following technologies:

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend Framework:** Flask (Python)
* **Database:** MySQL
* **QR Code Generation:** Python QRCode Library
* **PDF Generation:** ReportLab
* **Authentication & Security:** Flask Sessions, Werkzeug Password Hashing
* **QR Scanner Module:** HTML5-QRCode JavaScript Library
* **Operating System:** Windows / Linux
* **Development Environment:** Visual Studio Code
* **Version Control:** Git & GitHub

The system follows a modern web application architecture where Flask manages the business logic, MySQL handles persistent data storage, ReportLab generates downloadable PDF tickets, QR Code technology enables ticket validation, and HTML/CSS/JavaScript provide a responsive user interface for passengers and administrators.
